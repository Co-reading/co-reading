## Getting Started

* Install git https://git-scm.com/
* Install python https://www.python.org/
* Install co-reading

``` python
pip install -U co-reading
```

* Add your book

``` python
co-reading add book
```

* Start reading

``` python
co-reading read
```

* Pause reading

``` python
co-reading pause
```
